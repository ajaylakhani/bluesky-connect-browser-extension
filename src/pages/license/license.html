Important Notice

Bluesky Connect is an independent third-party extension and is not officially affiliated with, endorsed by, or connected to Bluesky PBC. All Bluesky logos, trademarks, and brand elements are the property of Bluesky PBC.

This extension accesses the Bluesky network via the AT Protocol and is subject to the Bluesky Terms of Service. Use of this extension to access Bluesky services constitutes acceptance of those terms.

When using this extension, your use of the Bluesky service remains governed by Bluesky's Terms of Service and Privacy Policy.

Bluesky Connect is designed to facilitate secure authentication with the Bluesky social network using the AT Protocol. This extension:

  • Stores authentication tokens securely in your browser
  • Only communicates with official Bluesky API endpoints
  • Does not collect or transmit user data to third parties
  • Complies with Bluesky's API usage guidelines

The developer of this extension reserves the right to modify it to maintain compatibility with changes to the Bluesky API or AT Protocol.
