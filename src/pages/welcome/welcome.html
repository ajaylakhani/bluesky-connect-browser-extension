Why Use App Passwords?

Why You Should Be Using Bluesky's 'App Passwords'

Securely enhance your Bluesky experience.

Bluesky has emerged as a developer-friendly social platform that offers robust API functionality. The platform has become particularly popular among former X users seeking an alternative social media experience. For everyday users, this means access to a variety of third-party applications that enhance how you interact with your Bluesky feed, publish content, expand your network, and more. Bluesky's dedicated community showcase features numerous client applications developed using their API. While these applications can work with your primary Bluesky credentials, creating dedicated app passwords provides an additional layer of security.

Understanding Bluesky App Passwords

To facilitate seamless integration with community-developed projects, Bluesky has implemented a system of app passwordsâ€”specialized access keys that grant third-party applications permission to interact with your account while maintaining security. These individual keys can be revoked anytime, giving you greater control without compromising your main account password.

Each app password functions as a unique authorization code generated specifically for connecting your Bluesky account to external applications. After creating an app password, you'll need to enter it into the third-party client to establish the connection (important note: Bluesky doesn't store these passwords for later viewing, so consider saving it somewhere secure if needed). You have the flexibility to create multiple app passwords for different services, and you can easily delete any password to immediately terminate that application's access to your account.

How to create app passwords in Bluesky

Follow these steps to create a Bluesky app password:

  1. Open Bluesky and go to the settings page. On desktop, the Settings button is in the left pane, and in the BlueSky App, it's under the hamburger menu (three horizontal lines) in the top-left corner.
  2. Navigate to Privacy & Security > App passwords. Alternatively, you can directly go to the app passwords page here.
  3. Select Add App Password.
  4. Choose a name for your app password. Ideally, you want to use a name that you can recognize, such as, "Deck Blue," which tells you exactly which app is using that app password.
  5. On the same page, you'll see a checkbox labelled Allow access to your direct messages. Think carefully before enabling this option. If a third-party app goes rogue, you want it to have access to as little of your information as possible. If you're using a third-party app as your primary Bluesky client, you might want to give it access to your DMs, but if you're just trying out a new client or using it only for a different feed view, you can avoid this.
  6. Once you're ready, click Next.
  7. You'll see a new app password on the screen. Copy it and paste it onto the login page for your desired third-party Bluesky client.

That's it! You can repeat this process to generate more app passwords. To revoke access for the app using the password, just click the red trash icon next its app password and select Delete.
